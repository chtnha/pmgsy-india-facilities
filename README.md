# PMGSY India Rural Facilities
This is the effort to visualise rural facilities released by Pradhan Mantri Gram Sadak Yojana (PMGSY) http://omms.nic.in/ in R.

* To demonstrate the R functions, I have only visualied facilities in Karnataka state. You can see the distribution of different categories (Agro, Education, Medical, Transport/Admin) of facilities in the below image:

![](https://github.com/chtnha/pmgsy-india-facilities/blob/main/Output%20Images/only-facility-categories.png)

* Overall facilities visulised with title & labels using ggplot & geom_sf packages

![](https://github.com/chtnha/pmgsy-india-facilities/blob/main/Output%20Images/karnataka-facilities.png)
